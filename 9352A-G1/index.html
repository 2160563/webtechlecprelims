World Wide Web (WWW)

What is WWW?

  • It is the collection of web resources and applications via the internet.
  • Way of information exchange through the internet.
  • A universe of network-accessible information.

Sir Tim Berners-Lee

  • Is an English engineer and computer scientist who invented the WWW in 1989.
  • Was the first to apply a working connection between a server and a Hypertext Transfer Protocol (HTTP) client using the internet.
  • He is the director of the World Wide Web Consortium (W3C).
  • Was awarded the Queen Elizabeth Prize for Engineering on March 18, 2013 for a "ground-breaking innovation in engineering that has been of global benefit to humanity".

Uniform Resource Locator(URL)

  • Popularly known as the web address
  • This is how we address a web resource that points out its location in a computer network and a way on how to access it.

http:// webtek.org / index.html

Protocol

- is a standard used in defining a method of exchanging data.

Domain Name

- is a string that represents the IP address of the resource.

Resource Name

- the file that is currently being accessed.
