CSS Preprocessors

CSS Preprocessors are designed to be a simple language and it generates CSS using a custom language syntax that typically includes features that donâ€™t exist in pure. SASS, LESS and Stylus are the well-known CSS Preprocessors that you can use. In addition, CSS Preprocessors is its own language built on top of CSS that adds scripting functionality.

CSS Frameworks

CSS Frameworks is a prewritten style sheet that includes its own layout and theme for user interface elements and it also provides predefined CSS design functionality that can be reused, extended or customized. In addition, CSS Frameworks has a prewritten style for known design elements across the network. The most popular and used frameworks are Bootstrap, Foundation and Materialize that you should pay a little attention.

CSS Polyfills

Polyfills shows ways of detecting certain features that cannot be supported by a browser and it provides predefined CSS.
