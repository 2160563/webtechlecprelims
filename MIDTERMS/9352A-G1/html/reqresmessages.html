HTTP Request & Response Messages

Request Message

  • The request message from the client to the server
  • First line: Request Line
    • Syntax: METHOD_TARGET_PROTOCOL
    • Note: Please keep in mind that the underscores (_) are considered as SP or spaces to separate the codes from one another. 
            GET /doc/home.html HTTP/1.1
            Host: www.webtek.org
            Accept: image/gif, image/jpeg, */*
            Accept-Language: en-us
            Accept-Encoding: gzip, deflate
            User-Agent: Mozilla/4.0
            Content-Length: 35

Response Message

  • After the request message has been sent and interpreted, the server will now respond with an HTTP response message
  • First line: Response Line
    • Syntax: PROTOCOL_STATUSCODE_TEXTUALPHRASE
    • Note: Please keep in mind that the underscores (_) are considered as SP or spaces to separate the codes from one another. 
            HTTP/1.1 200 OK
            Date: Sun, 04 March 2018 08:18:14 GMT
            Server: Apache/1.3.29 (Win32)
            Last-Modified: Sat, 24 Feb 2018
            ETag: "0-23-4024c3a5"
            Accept-Ranges: bytes
            Content-Length: 35
            Connection: close
            Content-Type: text/html
