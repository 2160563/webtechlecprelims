JS Client-Side Execution Environment

Client-Side JS

  • is an extended version of JavaScript that enables the enhancement and manipulation of web pages and client browsers
  • it is dependent on the core JavaScript and cannot work without it.

Core JavaScript (ECMAScript)

  • the base JavaScript language
  • ECMASCRIPT is the standard for scripting languages
  • JavaScript and other scripting languages use ECMAScript as their core

(Traditional) Browser Object Model (BOM) API

  • non-standardized meaning browser verndors are free to implement the BOM in any way they wish.
  • is a heirarchy of browser objects used to manipulate properties and methods related to the browser itself
  • Objets include: window, navigator, screen, location, history, etc.

Document Object Model (DOM) API

  • DOM API Interfaces
    • Node, Document, Element, Text, Attr, DocumentType, Comment, etc.
    • Event, EventTarget, EventListener, etc.
    • CSSStyleSheet, CSSRule, etc.

Misc. JavaScript Web APIs

  • Geolocation, IndexedDB, Local Storage, File, Push Notifications, Service Workers, Web Sockets, Web Workers, XMLHttpRequest (AJAX), etc.
Most of the APIs are accessed from scripts via the global object 'Window'
