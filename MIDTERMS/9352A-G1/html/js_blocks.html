Basic JS Building Blocks

  1. Conditionals

    2. As with any other programming languages, decision making capabilities are a necessity. This is basically how they control the flow of the program. JavaScript has such a simple function.

    <script language = "JavaScript">
var first_name = "Luger";

if (first_name == "Denmark") {
    document.write("First IF " +first_name);
}
else if (first_name == "Jasper") {
    document.write("Second IF " + first_name);
}
else if (first_name == "Dom") {
    document.write("Third IF " +first_name);
}
else {
    document.write("ELSE PART " + first_name);
}
</script>
  2. Looping

    3. Looping is something that you need when you want something done more than once, JS obviously supports this kind of capability.

    //for loop
for (var i = 0; i < 20; i++) {
    alert(i);
}

//while loop
var n = 1;
while (n < 5) {
    console.log("Looping");
    n++;
}
  3. Functions

    4. Functions can be considered as the definitive element of JavaScript. They are code written in a single block of code which is designed to perform a specific task. In Java terms, they are a bit similar to methods but are a bit more flexible.

    Functions can be easily called to do its required tasks. They can also be defined by the users themselves.
