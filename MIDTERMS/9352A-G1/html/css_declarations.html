CSS Declarations

What is CSS Declaration?

A CSS declaration sets CSS properties to identified values is the core function of the CSS language and any CSS engine calculates which declarations are able to apply in every single element of a page in order to appropriately style it and lay it out.

Properties

  • Shorthand Properties

    • These properties allow authors to specify the values of several properties with a single property.

    Examples: 
    background, font, margin, padding, border
  • Vendor Specific Extensions (also known as vendor prefixes)

    • Used by browser vendors as a prefix for the names of experimental or non-standard CSS properties; lately, vendors are moving away from vendor prefixes in favor of user-controlled flags or preferences

    Examples:
    -webkit, -moz, -o, -ms
  • Custom Properties, also known as CSS variable (experimental)

    • Property names prefixed with --, representing a value that can be reused throughout the document using the var() function.

Value Types

  • Keywords

    • A CSS keyword is a pre-defined value that can be used as the value of a CSS property.

    CSS-Wide Keywords

    Here are the keywords that can be used for any CSS property value:

    inherit

    This keyword copies the value of the property in effect from the parent of the element.
    initial

    This keyword represents the default value of the property.
    unset

    This keyword is same as the inherit keyword. It allows a property to adapts the inherited value from the parent element, otherwise set to initial value if there is no parent at all.
  • Numbers

    • These are simply integers or reals in (scientific) decimal notation.

  • Dimensions
    • Length Units
      • Font-relative 
      em, ex, ch, rem
      Viewport-percentage 
      vw, vh, vmin, vmax
      Absolute 
      cm, pt, pc, px, in
    • Angle Units

      • These units are used in gradient and transform functions.

      deg, grad, rad, turn
    • Duration

      • Used in animations, transitions and other related properties.

      s, ms
    • Frequency Units

      • These values are denoted by <frequency>, and are used with aural CSS like the pitch of a speaking voice. It initially introduced in CSS2 for the (obsoleted) aural media types; reintroduced in CSS3, but is currently unused

      Hz, kHz
    • Resolution Units

      • These values are used in media queries.

      dpi, dpcm, dppx
    • Percentages

      • These are numbers with a percent sign (%) suffix. Calculated as a percentage of some value (usually taken from the parent element).

    • URLs and URIs

      • These are url() function with an absolute or relative (to the stylesheet) URL parameter. It demotes a pointer to a resource, such as an image or a font.

    • Colors
      • Basic Color Keywords

        • These are simply the colors names, which are case-insensitive.

        aqua, blue, green, red, silver, gray
      • RGB/RGBA hexadecimal notation: #RGB[A]

        • R (red), G (green), B (blue), and A (alpha) are hexadecimal characters (from 0 up to 9, and from A up to F). A (alpha) is optional. The three-digit notation (#RGB) is a shorter version of the six-digit form (#RRGGBB), and the four-igit notation is a shorter version of the eight-digit for (#RRGGBBAA). 

        #ffffff, #000000, #cdcdcd, #0000ff
      • RGB/RGBA Function: rgb[R, G, B] or rgb[R, G, B, A]

        • The R, G, and A can be either integers or percentage, where the 255 value is equivalent to 100%. Since alpha (A) corresponds to transparency and opacity, the accepted value ranges between 0 and 1 or 0% to 100%, where 1 is equivalent to 100% opacity and 0 for transparency.

        rgb(255,0,153), rgb(255, 0, 153, 1)
      • HSL Function: hsl[H, S, L] or hsl[H, S, L, A] or hsla[H, S, L, A]

        • Using this function, a color can be defined according to its hue, saturation and lightness using the hsl() and hsla functions.

        Hue (H) is an angle of the color circle, so it means that the value in H is in number or integer.

        Saturation (s) and Lightness (L) are percentages, where 100% leads to completely saturation and white for lightness.

        Alpha (A) is always optional. It can be a number between 0 and 1 or 0% to 100%, where the number 0 is equivalent to 100% transparency.

    • Strings

      • Delimited by single quotes (‘) or double quotes (“)

    • Functions

      • Used as a value for various CSS properties

      attrib(), var(), rgb(), hsl()
    • Miscellaneous Types

      • CSS properties that are highly experimental or don’t fit in any other categories.
