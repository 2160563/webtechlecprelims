JavaScript

What is JavaScript?

JavaScipt is a client-side scripting language used to add interactivity to html and css files. It was created by Brendan Eich in 1995. JavaScript was originally developed under the name "Mocha" but to jump onto the ‘Java’ bandwagon the developers decided to replace it with the name ‘JavaScript’.

Using JavaScript, complex functionality can be implemented on a website. Along with HTML and CSS, JavaScript is used to build interactive and responsive menus, maps, and other things for user interaction. Some of the website that accept variable text like in login, signup and registration uses JavaScript. Also, JavaScript is used for validating data, creating cookies, and detecting user's browser.

How JavaScript works?

  1. When the browser loads the website resources, it will fetch the pages with js code whether it is embedded onto a html file or on a separate .js file.
  2. Afterwards, the js code will then load. This does not necessarily mean that it will directly load/perform all the functions of the js file but it will wait for the necessary triggers or events defined before it can do so.
  3. Finally, the code is then executed line by line.
