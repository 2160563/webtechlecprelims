CSS Statements

What are Statements?

Statements are building blocks that begin with non-space characters and end with a semi-colon or first enclosing curly brace. There are two types of statements:

  1. CSS At-rules

    2. An At-rule statement starts with an at sign (@), and then followed by an identifier. Each type of at-rules may have its own internal syntax and semantics. These statements are used to convey conditional information, descriptive information and meta-data information. Here are some of examples of at-rules used in CSS:

    • @charset
    • @import
    • @namespace
    • @document
    • @font-face
    • @keyframes
    • @media
    • @page
    • @supports
  2. CSS Rule-sets

    3. CSS Rule-sets consists of selector(s) followed by an enclosed declaration block, which contains one or more declarations. Each declaration must end with semi-colon, otherwise the declarations will not be applied except the first declaration.
