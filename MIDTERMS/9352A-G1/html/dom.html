Document Object Model (DOM)

What is DOM?

  • is an Application Programming Interface (API) for HTML & XML documents
  • it allows access to the HTML document from within scripts associated with the web page
  • a parsed HTML document is represented by a DOM tree, which contains nodes representing elements, element attributes, textual content, and other HTML document components e.g: DOCTYPE, comments, etc.
  • the API is accessed via the Document object property of the global Window object.
  • it provides a standard programming interface that can be used in a wide variety of environments and applications since it is a W3C specification.

DOM Functionalities

  • Retrieve references to nodes or (node collections) in the DOM:
getElementById()

returns the element that has the specified value of the ID attribute
getElementsByTagName()

returns all the elements that has the specified tag name
getElementsByClassName()

returns all the elements that has the specified class name
getElementsByName()

returns all the elements that has the specified name (the value of the name attribute)
querySelector()

returns the first element that matches a specified CSS selector
querySelectorAll()

returns all the elements that matches a specified CSS selector
  • Traverse the DOM tree (from a given node:)
parentNode

returns the parent node of the specified node as a Node object
parentElement

returns the parent element of the specified element
childNodes

returns all of the node's child nodes as a NodeList object
children

returns all of an element's child elements as an HTMLCollection object
firstChild

returns the first child node of the specified node as a Node object
lastChild

returns the last child node of the specified node as a Node object
nextSibling

returns the next node after the specified node in the same tree level
previousSibling

returns the previous node after the specified node in the same tree level
firstElementChild

returns the first child element of the specified element
lastElementChild

returns the last child element of the specified element
nextElementSibling

returns the next element after the specified element in the same tree level
previousElementSibling

returns the previous element after the specified element in the same tree level
  • Construct/copy nodes:
createElement()

creates an element node with the specified name
createTextNode()

creates a text Node with the specified text
createAttribute()

creates an attribute with the specified name and returns the attribute as an Attr object
cloneNode()

creates a copy of a node and returns the clone
importNode()

imports a node from another document
  • Manipulate the DOM tree:
appendChild()

appends a node as the last child of the node
insertBefore()

inserts a node as a child right before an existing child that you specified
replaceChild()

replaces a child node with a new node
removeChild()

removes a specific child node of the specified element
adoptNode()

adopts a node from another document

  • Misc. Attributes and Methods
nodeType

returns the node type as a number of the specified code
nodeName

returns the name of a specific node
nodeValue

sets or returns the node value of the specific node
attributes

returns the specified node's attributes
id

returns the id of an element

tagName

returns the tagName of an element
className

returns the className of an element
classList

returns the className's of an element as a DOMTokenList object
ownerDocument

returns the owner document of a node, as a Document object
documentElement

returns the documentElement as an Element object
hasChildNodes()

returns true if the node has any child nodes, if none, returns false
normalize()

removes empty text nodes and joins adjacent text nodes
  • Manipulate attributes (for Element nodes):
setAttribute()

adds the specified attribute to an element then gives it the value
getAttribute()

returns the value of the attribute
removeAttribute()

removes the attribute from an element
setAttributeNode()

adds the specified attribute node to an element
getAttributeNode()

returns the attribute node with the name of the element as an Attr object
removeAttributeNode()

removes the attribute from an element then returns the attribute that was removed
  • Event Handling:
addEventListener()

attaches an event handler
removeEventListener()

removes an event handler
dispatchEvent()

dispatches an event and invokes the EventListeners that it possesses
  • Style Handling:
styleSheets

returns a StyleSheetList of CSSStyleSheet objects
ownerRule

returns CSSImportRule, else returns null if not imported into the document
cssRules

returns a live CSSRuleList
insertRule()

inserts a new rule in the stylesheet
deleteRule()

deletes a rule in the stylesheet
getComputedStyle()

gets all the CSS properties and values
